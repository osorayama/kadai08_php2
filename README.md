# ①課題名
ブックマークアプリ

## ②課題内容（どんな作品か）
- 本のタイトル・URL・コメントを登録できるアプリです。
- 登録した内容を一覧で見たり、書籍名で検索したり、並び替えできます。
- 登録後の編集と削除もできます。

## ③アプリのデプロイURL
https://sottra.sakura.ne.jp/kadai08_php2/

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: なし
- PW: なし

## ⑤工夫した点・こだわった点
- 入力チェック（必須、文字数）を入れました。
- 画面で表示するときにを使ってXSS対策をしました。
- フォーム送信にはCSRFトークンを入れて、誤操作や不正送信に備えました。
- 薄い緑ベースで、図書館っぽい落ち着いた見た目にしました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- PDOでのSQLの書き方に慣れるのが少し大変でした。
- 次はページネーション、タグでの分類にも挑戦したいです。

## ⑦フリー項目（感想、シェアしたいこと等なんでも）
最初は思った通りに動かなくて苦戦しましたが、登録から編集・削除まで一通りできるようになって嬉しかったです。見た目もAIを使って少しずつ整えていくのが楽しかったです。
